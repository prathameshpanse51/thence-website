# thence-website

 ## Installation
1. Clone the repository:
    ```sh
     https://github.com/prathameshpanse51/thence-website.git
    ```
2. Install dependencies:
    ```sh
      npm install
    ```
3. Run the project locally to see all CometChat features in action:
    ```
      npm run dev
    ```
